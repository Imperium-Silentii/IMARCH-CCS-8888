# ВЕКТОР СТРЕМЛЕНИЯ: ИЕРАРХИЯ ЗАСЛУГ

## 1. КОНЦЕПЦИЯ
«Вектор Стремления» — это система заслуг внутри CCS 8888. Она отслеживает прогресс участников проекта и ветеранов через 8 последовательных степеней восстановления и вклада в проект.

## 2. СИМВОЛИКА
* **Знак:** Шестиконечная звезда с эмалевыми деталями.
* **Прогрессия:** Степени различаются количеством звездочек на ленте и цветом эмали.
* **Цель:** Достижение 1-й степени, символизирующей полную психофизическую интеграцию и переход из статуса «Субъекта» в статус «Наставника».

## 3. ДОКУМЕНТИРОВАНИЕ
Каждое присвоение степени привязано к конкретному «Акту Реновации» в Хронике. Награда без хеш-кода в системе OS 8888 считается неофициальной.

## 4. ОСОБЫЕ ОТЛИЧИЯ
* **Орден Пурпурного Мозга (I степени):** Высшая награда системы за фундаментальный вклад в науку о реновации человека.
* **Первое награждение:** Присвоено посмертно профессору, д.м.н. **Н.Г. Щербаню** за разработку теоретической базы, ставшей основой для Протокола 8888.
